What is Mininet

Mininet is a network emulator, or perhaps more precisely a network emulation orchestration system.It runs a collection of end-hosts, switches, routers, and links on a single Linux kernel. It uses lightweight virtualization to make a single system look like a complete network, running the same kernel, system, and user code. Mininet provides cool Python API.

Dependent Writing

Posted on 2019-09-30 Edited on 2019-10-04 In Writing strategy

准备托福，需要需要提高写作能力，这也是成为一个优秀的PhD Student 的必要条件。

首先说说托福写作的第一部分，这一部分相对简单，主要是从阅读材料和听力材料中找出作者的观点。首先判断两部分作者之间的态度，支持还是反对。一般听力材料都会针对阅读材料中每一个观点进行分析，阐述支持的理由或者反对的理由。这个Question中信息的收集难度不大，要提高成绩是在以信息的展示能力。在Response中我们的结构一般都是先说两部分材料作者的观点是相合还是相斥，然后逐条解释原因。在逐条解释中，一般都需要同时提到阅读和听力中的观点，这个时候，观点的组织能力和表达能力就很重要了。

为什么要用cmake

CMake和AutoTools之类的是make系统上层的的大型项目构建工具，通过其可以编写跨平台的配置文件，用于生成底层的Makefile。Makefile具有平台依赖性，一般会基于特定的编译环境来构建flow，而且手动编写Makefile的工作量比较大，解决依赖关系的时候也容易出错。
